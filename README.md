img2aa

画像を読み込み、高精度な輝度計算とガンマ補正を用いて、Windowsコンソール上に最適化されたアスキーアートを表示するC言語製のツールです。

特徴

70段階の密度スケール: 豊富な文字セットを使用し、写真のディテールを細部まで再現します。


自動コントラスト調整: 画像内の最小・最大輝度をスキャンし、コントラストを自動で最適化します。


ガンマ補正機能: pow()関数による補正（1.6）を適用し、暗部のつぶれを防ぎ、写真らしい階調を維持します。



アスペクト比補正: ターミナルの行間に合わせて垂直方向を微調整し、歪みの少ない表示を実現します。


UTF-8対応: Windows環境での文字化けを防ぐため、コンソールの出力コードページを自動でUTF-8（65001）に設定します。


動作要件


OS: Windows 10/11


コンパイラ: GCC (MinGW-w64等)


依存ライブラリ: stb_image.h（画像読み込み用）


セットアップとビルド

stb_image.hをソースファイルと同じディレクトリに配置します。


以下のコマンドでコンパイルします：

Bash

gcc main.c -o img2aa.exe -lm

※数学関数を使用するため、環境によっては -lm オプションが必要です。

使い方

img2aa.exeを実行します
。

変換したい画像ファイルのパスを入力します（ドラッグ＆ドロップによる入力にも対応しています）。


コンソール上にアスキーアートが表示されます。


Tips: ターミナルのフォントサイズを小さく設定すると、より高精細な表示を楽しむことができます。


技術詳細

最大幅: デフォルトでは200文字に設定されています（MAX_WIDTHで調整可能）。


アルゴリズム:


stbi_loadで画像をグレースケールとして読み込み。


画像の最小・最大輝度を特定し、正規化。


corrected = pow(normalized, 1.6) によるガンマ補正。


文字密度配列 G_SCALE から最適な文字を選択。


ライセンス
このプロジェクトは、画像読み込みにパブリックドメインの stb_image.h を使用しています。
